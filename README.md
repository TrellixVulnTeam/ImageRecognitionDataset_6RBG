ImageRecognitionDataset
====

Caltech101/256, CIFAR-10/100, MNIST/FashionMNIST, omniglot

## Requirement
python 3.x

## Install
### pip
```bash
$ pip install numpy
$ pip install pillow
```
### poetry
```bash
poetry install
```

## Usage
```bash
$ python main.py {CIFAR10, CIFAR100, MNIST, fashionMNIST, caltech101, caltech256, omniglot}
```
